# SIGESBM - Sistema de Gestão de Escalas
## Padrões e Diretrizes do Projeto (Versão Simplificada)

**Data de Criação:** 2 de março de 2026  
**Versão:** 2.0 - TCC Essencial  
**Tipo:** TCC - Análise e Desenvolvimento de Sistemas (ADS)

---

## 📌 VISÃO GERAL RÁPIDA

**O que é:** Sistema web para gerenciar escalas de trabalho  
**Stack:** Laravel 10 + Vue.js 3 + MySQL 8 + Docker  
**Tempo:** 3 meses  
**Foco:** Features essenciais, código limpo, funcional

### Funcionalidades Principais
- Login/autenticação de usuários
- Criar e gerenciar escalas de trabalho
- Atribuir funcionários a turnos
- Visualizar calendário de escalas
- Gerar relatórios simples
